# Logical Operators

There are 3 basic logical operators,

| Operators       | Example/Description                                          |
| --------------- | ------------------------------------------------------------ |
| && (AND)        | (x>5)&&(y<5) - It returns true when both conditions are true |
| Shift + \\ (OR) | (x>5)|| (y>5)                                                             |
